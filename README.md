# 🏭 FMCG Lakehouse Consolidation — Databricks ETL Pipeline

Real-world data engineering project: A large FMCG parent company acquires a smaller sports nutrition brand ("Sports Bar"). This pipeline ingests, cleans, and consolidates the child company's data into the parent's unified Lakehouse using Databricks, Delta Lake, and Amazon S3.
